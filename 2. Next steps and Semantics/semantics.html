semantics refers to using proper tags instead tags instead of divs and spands for everything

examples of semantics

  1. main - used to wrap the main content
  2. nav - used to show elements with links to other parts of page or other pages
  3. section - when there are no other proper semantic element
  4. article - content that can be shared with others, should have a heading
  5. aside - not the main content but related
  6. header - main top part of webpage
  7. footer - the bottom portion
  8. time - to wrap time
  9. figcaption - to add a caption for images
